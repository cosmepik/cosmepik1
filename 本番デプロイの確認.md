# 本番デプロイの確認リスト

GitHub にプッシュすると Netlify が自動でデプロイします。以下を確認してください。

---

## 1. Netlify の環境変数

**Site configuration** → **Environment variables** で次が設定されているか確認：

| Key | 設定 |
|-----|------|
| `NEXT_PUBLIC_SUPABASE_URL` | あなたの Supabase URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | あなたの Supabase Anon Key |
| `RAKUTEN_APPLICATION_ID` | （楽天検索を使う場合） |
| `RAKUTEN_ACCESS_KEY` | （楽天検索を使う場合） |

※ Clerk を使わなくなったため、`NEXT_PUBLIC_CLERK_*` と `CLERK_SECRET_KEY` は削除してOKです。

---

## 2. Supabase のリダイレクト URL

認証メールのリンクや OAuth が動くよう、Supabase で本番 URL を登録してください。

1. **Supabase ダッシュボード** → **Authentication** → **URL Configuration**
2. **Site URL**: `https://cosmepikdemo1.netlify.app`（あなたの本番 URL）
3. **Redirect URLs** に追加:
   - `https://cosmepikdemo1.netlify.app`
   - `https://cosmepikdemo1.netlify.app/auth/callback`
   - `https://cosmepikdemo1.netlify.app/**`

---

## 3. デプロイ状況

- **Netlify** → **Deploys** で最新デプロイの状態を確認
- ビルドエラーが出た場合はログを確認
- 環境変数を追加・変更した後は **Trigger deploy** → **Deploy site** で再デプロイ

---

## 4. 動作確認

デプロイ完了後、以下を確認：

1. `https://cosmepikdemo1.netlify.app` にアクセス
2. 「サインイン / サインアップ」→ `/login` へ
3. メール＋パスワードでサインアップ
4. 確認メールが届く
5. メール内のリンクをクリックしてログイン完了
6. 管理画面（`/influencer/manage`）に遷移するか確認
